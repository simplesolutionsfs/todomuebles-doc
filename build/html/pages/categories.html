Categorías de Productos¶

Las Categorias son La clasificacion de los productos en Todomuebles.com, esto ayuda al comprador a encontrar el producto de una manera más facil y lógica. Cada producto que se importe al marketplace debe tener sus categorias correspondientes.

Nota

Para poder asignar las categorias a un producto puedes usar la tabla de abajo para entender a que categorias puede pertenecer.

Las categorias van en la columna categories en el archivo de importación. y separados por comas ( , ) para agregar multiples categorías.

Ejemplo:

Default Category/Muebles,Default Category/Muebles/Sala,Default Category/Muebles/Sala/Reclinables,Default Category/Muebles/Sala/Sillones Ocasionales

Este path asignara al producto las categorías de Default Category, Muebles, Sala, Reclinables y Sillones Ocasionales

Advertencia

Default Category es la categoria padre de todas las categorias, esta es la que se utiliza para agrupar las otras categorias y poder filtrar todos los productos del catalogo
